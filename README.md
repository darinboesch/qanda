# qanda

Question and Answer. Manage Question Sets and provide action processors for updates.

## Quickstart (Dev Mode)

```
  npm install
  npm run start
```

**Note : Please make sure your MongoDB is running.**

Navigate to localhost:3000 in your browser.

## Create Release Distribution

```
  npm run build
```
